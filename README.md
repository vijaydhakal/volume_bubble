A minimalist floating volume controller for Android.

## Features
- Floating bubble that snaps to screen edges.
- One-tap to show system volume slider.
- "Trash Zone" to dismiss the service.
- Light and Dark mode support.

##![Screen_recording_20260118_115309-ezgif.com-video-to-gif-converter.gif](../../Screen_recording_20260118_115309-ezgif.com-video-to-gif-converter.gif) Permissions Required
- `SYSTEM_ALERT_WINDOW`: To display the bubble over other apps.
- `FOREGROUND_SERVICE`: To keep the controller active.

## How to Build
1. Clone the repo.
2. Open in Android Studio.
3. Build and Run on a device with API 26+.
